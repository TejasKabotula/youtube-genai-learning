# YouTube Tutor - Video Summarization & Learning System

An AI-powered MERN application for video summarization, question generation, and doubt clarification.

## Features
- **YouTube Integration**: Extract transcripts from any YouTube video.
- **AI-Powered Summaries**: Short, Medium, and Detailed summaries generated by LLM.
- **Intelligent Question Generation**: MCQs, open-ended, and short-answer questions with difficulty levels.
- **Ambiguity Detection**: Automatically detects unclear video segments and provides detailed explanations.
- **Interactive Doubt Chat**: Get instant answers to your specific doubts using RAG.
- **Educator Dashboard**: Review, edit, and export quiz content as PDF or DOCX.
- **Multilingual Support**: English, Telugu, and Hindi.

## Tech Stack
- **Frontend**: React, Vite, TypeScript, Tailwind CSS, Lucide Icons, Framer Motion.
- **Backend**: Node.js, Express.js, TypeScript, Mongoose.
- **Database**: MongoDB.
- **DevOps**: Docker & Docker Compose.

## Getting Started

### Prerequisites
- Docker & Docker Compose
- LLM API Key (OpenAI compatible)

### Setup & Run
1. **Clone the repository**:
   ```bash
   git clone <repo-url>
   cd youtubevideo
   ```

2. **Configure Environment Variables**:
   Create a `.env` file in the `server` directory based on `.env.example`:
   ```bash
   PORT=5000
   MONGODB_URI=mongodb://mongo:27017/youtube-tutor
   JWT_SECRET=your_jwt_secret
   LLM_API_KEY=your_actual_api_key
   LLM_MODEL=gpt-4-turbo
   ```

3. **Run with Docker**:
   ```bash
   docker-compose up --build
   ```

4. **Access the App**:
   - Frontend: `http://localhost:3000`
   - Backend API: `http://localhost:5000`
   - MongoDB: `mongodb://localhost:27017`

### Local Development (without Docker)
1. **Install root dependencies**:
   ```bash
   npm install
   ```
2. **Setup Server**:
   ```bash
   cd server
   npm install
   # Configure .env (use localhost for MongoDB URI)
   npm run dev
   ```
3. **Setup Client**:
   ```bash
   cd client
   npm install
   npm run dev
   ```

## Project Structure
- `/client`: React application with Vite.
- `/server`: Express API with TypeScript.
- `/server/src/models`: Mongoose schemas.
- `/server/src/services`: Core logic (AI, YouTube, Export).
